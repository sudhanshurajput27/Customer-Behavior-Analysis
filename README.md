This Repo is a practice project I worked on in 2025 with the help of tools like:-
1) Python - Pandas, NumPy, MatplotLib
2) PowerBi
3) MySQL

The dataset I used was from an e-commerce site about customer behaviour, so I took that dataset and analysed it
using Python, which helped me clean, visualise and find the missing errors in this dataset, then after cleaning 
The data I moved onto transfer to MySQL and ran some queries to find the results, helping in analysing the data
for the best future marketing, and then after running those queries, I made a dashboard using PowerBi.
